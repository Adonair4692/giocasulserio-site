# Checklist di pubblicazione

## Contenuti

- [ ] Titoli italiani ed inglesi controllati.
- [ ] Attribuzione di ogni progetto verificata.
- [ ] “Istituzionale” e “indipendente” usati correttamente.
- [ ] CEFLI sciolto come Centro di Formazione Logistica Interforze.
- [ ] Civil Discourse non presente.
- [ ] Nessun dato riservato o personale nei testi e nei commenti.
- [ ] Numeri manuali compilati soltanto con valori difendibili.

## Link

- [ ] Ogni link Gioca, Consulta e Acquista apre il sito corretto.
- [ ] Nessun URL inventato.
- [ ] Link esterni con `target="_blank"` e `rel="noopener noreferrer"`.
- [ ] Email `info@giocasulserio.it` provata.
- [ ] LinkedIn e ORCID provati.

## Immagini e diritti

- [ ] Registro asset aggiornato.
- [ ] Fotografia personale originale o autorizzata.
- [ ] Loghi istituzionali autorizzati.
- [ ] Fotografie con persone pubblicabili.
- [ ] Copertine e documenti con attribuzione corretta.

## UX e accessibilità

- [ ] Hero visibile subito senza spazio vuoto.
- [ ] Test a 320, 360, 390, 768, 1024 e desktop.
- [ ] Nessuno scroll orizzontale.
- [ ] Menu mobile funzionante.
- [ ] Navigazione da tastiera.
- [ ] Focus visibile.
- [ ] Popup chiudibili con tastiera.
- [ ] Suoni disattivati inizialmente.
- [ ] Modalità reduced motion verificata.
- [ ] Contrasti verificati.

## Funzioni

- [ ] Ricerca: Pompei.
- [ ] Ricerca: wargame.
- [ ] Ricerca: emergenza.
- [ ] Ricerca: soft skills.
- [ ] Ricerca senza accenti.
- [ ] Filtri progetti.
- [ ] IT/EN.
- [ ] Black Swan Score.
- [ ] Scadenza sette giorni.
- [ ] Reset percorso.
- [ ] Platinatura.
- [ ] Pagina 404.

## Privacy e sicurezza

- [ ] Privacy Policy verificata.
- [ ] Cloudflare Web Analytics attivato e descritto.
- [ ] Nessuna API key o password nel repository.
- [ ] `_headers` pubblicato da Cloudflare Pages.
- [ ] CSP controllata dopo l’attivazione di servizi esterni.
- [ ] Nessun modulo o tracker aggiunto senza aggiornare informativa e CSP.

## DNS e posta

- [ ] Esportazione o screenshot dei record DNS.
- [ ] Record MX conservati.
- [ ] SPF conservato.
- [ ] DKIM conservato.
- [ ] DMARC conservato.
- [ ] Invio e ricezione email verificati prima e dopo il cambio sito.
- [ ] SiteBuilder disattivato soltanto dopo il collaudo del nuovo sito.
